# VencedoresPiorFilme

### Execução do Teste

Pelo *cmd* abra a pasta raiz do projeto 

`mvnw test`

### Compilando o projeto
Pelo *cmd* abra a pasta raiz do projeto

`mvnw package`

### Execução do projeto
Pelo *cmd* abra a pasta raiz do projeto

`mvnw spring-boot:run`

Ou, caso já tenha compilado, pode-se também executar o *VencedoresPiorFilme-0.0.1-SNAPSHOT* criado dentro da pasta *target*.
